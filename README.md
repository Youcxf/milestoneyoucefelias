## Getting Started

You can download the backend from https://github.com/Youcxf/milestone1 or you can just use the base URL https://milestone1-ogfx.onrender.com/

### Prerequisites
* **Node.js** 
* **npm** 

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/Youcxf/milestoneyoucefelias.git](https://github.com/Youcxf/milestoneyoucefelias.git)
    ```

2.  **Navigate to the project directory**
    *Note: The project source is located inside the subfolder.*
    ```bash
    cd milestoneyoucefelias/milestoneyoucefelias
    ```

3.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

4.  **Start the development server**
    ```bash
    npm start
    ```
    Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
